# Banking Customer Analytics

## Project Overview

This project analyzes **10,000 banking customer records** using SQL Server and Microsoft Power BI to uncover customer behavior, financial trends, and business opportunities. The project provides valuable insights that support customer retention, segmentation, and strategic decision-making.



## Business Problem

Banks generate large volumes of customer data every day, but without proper analysis, it is difficult to understand customer behavior, identify high-value customers, reduce churn, and improve financial performance.

The objective of this project was to analyze customer data and develop an interactive dashboard that enables business stakeholders to monitor customer trends and make informed business decisions.



## Tools Used

- SQL Server
- Microsoft Power BI
- Microsoft Excel



## Data Cleaning Process

Before analysis, the dataset was cleaned and validated by:

- Checking for missing values
- Identifying duplicate records
- Validating customer information
- Verifying data consistency
- Preparing the dataset for SQL analysis



## SQL Analysis

SQL Server was used to analyze:

- Customer demographics
- Customer segmentation
- Account type analysis
- Income analysis
- Credit score analysis
- Regional analysis
- Balance analysis
- Customer churn analysis



## Power BI Dashboard

A **three-page interactive dashboard** was developed consisting of:

### Executive Summary
- Customer KPIs
- Total customers
- Customer churn rate
- Average account balance
- Revenue insights

### Customer Insights
- Customer demographics
- Income distribution
- Regional analysis
- Account type analysis
- Credit score insights

### Retention & Churn Analysis
- Customer churn trends
- High-risk customer segments
- Regional churn analysis
- Business recommendations

The dashboard enables stakeholders to monitor customer behavior, identify retention risks, and make data-driven decisions.



## Key Insights

- Analyzed **10,000 banking customer records**.
- Customer churn rate was **49.55%**.
- The **South** region recorded the highest customer losses.
- **Current account** holders experienced the highest churn.
- Male customers churned slightly more than female customers.
- Digital banking adoption was approximately **50.7%**, highlighting an opportunity for increased digital engagement.



## Business Recommendations

- Develop targeted customer retention campaigns for high-risk regions.
- Improve engagement with current account customers through personalized offers.
- Increase digital banking adoption through customer education and incentives.
- Use customer segmentation to deliver personalized financial products.
- Continuously monitor churn trends using interactive dashboards.
